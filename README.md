# Work EOD Project UI

**React Frontend for Work EOD Reporting System**

---

## 🔹 Project Description

This is the **frontend** of the Work EOD Project (ERN stack).  
It provides a user interface for employees to submit their **End-of-Day (EOD) reports**.  

**Tech stack:**  
- React  
- Node.js  
- Vite  
- PNPM  

**Key Features:**  
- Submit daily EOD reports
- Simple, responsive UI
- CI/CD deployment via GitHub Actions  

**Website:** [https://work-eod-report-ui.alconians.com](https://work-eod-report-ui.alconians.com)
